# Course Folder Index

This index lists the current Java learning-material folders in reading order. Paths are relative to the `LearnigMaterials` root.

## 00) START HERE

- `Complete Java Study Roadmap.md` - suggested study path.
- `Java Exam Cheat Sheet.md` - compact Java review.
- `All Markdown Files Index.md` - this navigation file.

## Main course folders

1) `Java Fundamentals` - JVM/JDK/JRE, variables, control flow, methods, arrays.

2) `Basics OOP Core` - classes, encapsulation, packages, static/final, records, enums, inner classes, object methods.

3) `Inheritance and Polymorphism` - constructors, `super`, abstract classes, interfaces, overriding, overloading, polymorphism, UML.

4) `Software Architecture and Design Patterns` - layers, facade, factory, repository, DTOs, snapshots, service layer, architecture mistakes.

5) `Generics` - generic classes, generic interfaces, iterators, wildcards, PECS, type erasure.

6) `Collections` - collection interfaces, `List`, `Set`, `Map`, `Queue`, sorting, frequency counting.

7) `Streams` - streams, lambdas, method references, collectors, grouping, partitioning, `Optional`.

8) `Exceptions` - checked/unchecked exceptions, custom exceptions, propagation, try-with-resources.

9) `Application Layers, Domain Workflows, and Authorization` - workflows, validation strategy, custom business exceptions, roles, audit fields, workflow testing.

10) `Java IO` - readers, writers, NIO `Path`/`Files`, scanners, formatters, serialization.

11) `Testing with JUnit` - assertions, lifecycle, testing patterns, examples.

12) `Git, Maven, Build Tools, and CI CD` - Git basics, Maven structure, `pom.xml`, dependencies, plugins, Maven Wrapper, CI, GitHub Actions.

13) `JDBC and Databases` - JDBC, `PreparedStatement`, transactions, database access basics.

14) `ORM JPA and Hibernate` - ORM, JPA, Hibernate, entities, relationships, cascade, fetch, lifecycle.

15) `EntityManager, Persistence Units, and Transactions` - `EntityManager`, persistence units, CRUD, transactions, rollback, H2.

16) `JPQL, TypedQuery, and Repository Pattern` - JPQL, typed queries, parameters, date ranges, aggregation, repositories, repository tests.

17) `Date and Time` - `LocalDate`, `LocalTime`, `LocalDateTime`, formatting, parsing, `Duration`, `Period`.

18) `Data Import, Logging, and Statistical Reports` - CSV import, validation, date parsing, import results, logging, reports, statistics, histograms, outliers.

19) `Final Revision Pack` - concise revision files, cheat sheets, question sets, traps, backend revision maps.

20) `Exercises` - standalone practice tasks for the full course, including the new backend topics.
