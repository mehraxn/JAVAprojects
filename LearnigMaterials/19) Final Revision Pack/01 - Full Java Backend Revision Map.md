## Revision map

```text
Java syntax
  ↓
OOP and inheritance
  ↓
Generics, collections, streams, exceptions, IO
  ↓
Testing and build tools
  ↓
JDBC, JPA, EntityManager, JPQL
  ↓
Architecture, workflows, imports, logging, reports
```

## Big idea

Backend Java is not only syntax. It combines:

- clear domain models;
- service-layer workflows;
- validation and authorization;
- data access;
- transactions;
- automated tests;
- repeatable builds;
- readable reports and logs.
