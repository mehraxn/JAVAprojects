# Java Learning Materials

Beginner-to-intermediate Java study notes, organized as a full course you can follow from
start to finish. Every topic uses the same style: a simple explanation, a worked example,
expected output, common mistakes, and exam-style reminders.

---

## What this folder contains

- **Study notes** for the whole language and its common tools, grouped into numbered folders
  `1)`–`13)` in learning order.
- **Exam revision material** in `14) Final Revision Pack` (cheat sheet, question sets, traps).
- **Hands-on practice** in `15) Exercises` (13 topic exercise files + one Stream lab).
- **Navigation help** in `00) START HERE` (this roadmap, the file index, and a cheat sheet).
- Some folders also include a `Code/` (or `Codes/`) subfolder with matching `.java` examples.

> **About the code examples:** all Java snippets are provided **for study**. They were checked
> by **static review only** (reading the code) and were **not compiled or executed** on this
> machine. If you have a JDK, you are encouraged to run them yourself and experiment.

---

## How to study the material

For each topic folder:

1. **Read** the explanation from top to bottom.
2. **Study the example** and its expected output; if you can, run it and change one line to
   predict the new result.
3. **Read the "common mistakes"** section — most exam marks are lost there.
4. **Do the matching exercises** in `15) Exercises`.
5. Before the exam, work through `14) Final Revision Pack`.

A good rhythm: one folder (or one lettered sub-topic) per study session, then its exercises.

## Which folders are which level

**Start here (foundations / basics)**
- `1) Java Fundamentals` — JVM/JDK/JRE, variables, operators, strings, control flow, methods, arrays.
- `2) Basics OOP Core` — classes, encapsulation, static/final, enums/records, inner classes, `equals`/`hashCode`.
- `3) Inheritance and Polymorphism` — constructors & `super`, overriding vs overloading, abstract vs interface, UML basics.

**Intermediate / more advanced**
- `4) Generics` — generic classes/interfaces, `Iterable`, wildcards & PECS, type erasure.
- `5) Collections` — `List`/`Set`/`Map`/`Queue`, `compute`/`merge`, sorting.
- `6) Streams` — streams, lambdas, method references, collectors, `Optional`.
- `7) Exceptions` — checked/unchecked, try-with-resources, custom exceptions.
- `8) Java IO` — readers/writers, NIO `Path`/`Files`, serialization.

**Applied topics and tools**
- `9) Testing with JUnit` — assertions, lifecycle, failure vs error.
- `10) Git and Build Tools` — Git basics/conflicts, Maven.
- `11) JDBC and Databases` — JDBC, `PreparedStatement`, transactions.
- `12) ORM JPA and Hibernate` — entities, relationships, cascade, fetch types.
- `13) Date and Time` — `LocalDate`/`LocalTime`/`LocalDateTime`, formatting, `Duration`/`Period`.

**For exam revision**
- `14) Final Revision Pack` — cheat sheet, mixed and topic-specific question sets,
  "predict the output" tracing, common traps, mistakes to avoid.

**For practice (use throughout the course)**
- `15) Exercises` — tasks with starter code, expected output, and full solutions, including
  "predict the output" and "fix the bug" questions for every major topic.
