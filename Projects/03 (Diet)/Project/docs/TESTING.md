# Testing

## Requirements

- Java 21
- Maven 3.9+ or the included Maven wrapper

## Run all tests

Linux/macOS/Git Bash:

```bash
./mvnw clean test
```

Windows PowerShell:

```powershell
.\mvnw.cmd clean test
```

Fallback if the wrapper is unavailable:

```bash
mvn clean test
```

## Scripts

Linux/macOS/Git Bash:

```bash
./scripts/test.sh
```

Windows PowerShell:

```powershell
.\scripts\test.ps1
```

Both scripts run `clean test` from the project root.

## Test coverage

The project includes the original example tests plus custom tests for:

- raw materials
- products
- recipes
- menus
- restaurants and opening hours
- customers
- orders
- takeaway queries
- validation
- end-to-end workflow

## JaCoCo report

Generate an HTML coverage report with:

```bash
./mvnw clean test jacoco:report
```

Windows:

```powershell
.\mvnw.cmd clean test jacoco:report
```

Report location:

```text
target/site/jacoco/index.html
```

The report is generated locally and should not be committed.

## Common failures

- If Java is older than 21, compilation can fail.
- If dependencies are not cached, the first Maven run may take longer.
- `target/` is generated by Maven and can be safely removed.
- Tests should not depend on execution order.
